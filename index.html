Automatische Tests für den JavaScript Hero

Diese handvoll WebSeiten sollen bei der Entwicklung helfen und ein schnelles Überprüfen der JavaScript-Funtionen auf der JS Hero Web-Site und auch lokal ermöglichen.

In diesem Projekt werden gegen die JavaScript-Funktionen von JS Hero Tests geschrieben. Mit Hilfe des Testrunner-Framework mocha ausgeführt.

Es gibt zwei Seiten, die die Ergebnisse anzeigen.

  1. Eine zeigt die Testergebnisse, wenn es eine lokale Installation gibt. Die Dateien der lokalen Installation müssen über ../jshero/ erreichbar sein.
  2. Die andere zeigt die Ergebnisse, wenn die Tests gegen die Web-Site www.jshero.net laufen.
